🎯 การดูข้อมูลปฏิทิน
• คลิกช่องวัน → ดูรายการงานทั้งหมดของวันนั้น แยกตามทีม
• Hover เม้าส์บน WO → Tooltip สรุปข้อมูลงานนั้น
• 🔴 หน้า WO = P1 Overdue (งานเร่งด่วนเกินกำหนด)
• 🔒 บน WO = Locked ย้ายไม่ได้ (มี Actual หรือ Closed)
📅 การ Reschedule งาน
• คลิก 1 ครั้ง บน WO → เปิดหน้าต่างอัปเดตสถานะ
• ดับเบิลคลิก บน WO → เปิด popup Reschedule ข้ามเดือนได้
• Drag & Drop WO ไปช่องวันอื่น → ย้ายภายในเดือน
⚠️ งานที่มี Actual Start/Finish หรือ Closed ห้ามย้าย

• Priority = กรองตาม P1-P4
• ← → เดือน หรือ Dropdown เลือกเดือน/ปี

🏭 CM Backlog รอ Shutdown — แยกตาม Plant

WO ที่ถูกตรวจพบเป็นงาน Shutdown (SD/ANSD/QSD/WS) แยกตาม Plant เพื่อดูว่าแต่ละโรงมีงานค้างรอ Shutdown กี่ WO (นับ Distinct)

📋 Shutdown Plan — ตรวจจับจาก Description & Status A-J (คลิกเพื่อกรอง)

ระบบสแกนข้อความใน Description ของทุก WO โดยอัตโนมัติ หากพบคำว่า SD (Shutdown), ANSD (Annual Shutdown) หรือ QSD (Quarterly Shutdown) จะดึงงานนั้นมาแสดงในหน้านี้ เพื่อให้ Planner เตรียมแผนล่วงหน้าด้าน Spare Part และงานจ้างได้ทันเวลา ก่อนวัน Shutdown จริง
